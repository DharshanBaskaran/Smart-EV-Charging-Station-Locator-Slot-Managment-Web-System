4. Software Used & Justifications

Software / Tech Description Justification
HTML5/CSS3/Vanilla JS Application PWA Environment Circumvents native browser popups with fully custom modals. Uses internal Service Workers for robust offline caching capabilities.
Node.js + Express High-concurrency Backend Modular routing efficiently divides domains (wallet, reviews, notifications, qr).
Socket.IO WebSocket Engine Allows instant telemetry & live progression updates for active charging sessions organically instead of using degraded HTTP polling.
MongoDB Atlas NoSQL Database Cloud Easily models shifting datasets like Ledger systems with advanced Mongoose schemas and cascading array validations.
bcryptjs / crypto Data Assurance Ensures passwords and security answers are aggressively hashed before DB entry.
Leaflet / Nominatim Map Analytics Renders interactive map markers, trip-polylines, and active geographic bounding filters entirely client-side.

5. Core Integrations & Testing

Crucial Validations Performed:

  • Dynamic Haversine Operations: Modifying vehicle battery range instantly updates strict radius limits across map features. PWA enforces limits reliably.
  • Offline Mode Preservation: Loading the platform URL without network connectivity immediately falls gracefully to cached `sw.js` handlers.
  • Trip Constraints: Plotted routes > 50km are allowed; attempting 30km ranges triggers Custom Low Battery Interruption Modals.
  • WebSocket Telemetry: Starting a session opens 2-way data flow mapping precise incremental percentage and cost (INR) into the floating Action view.
  • Immutable Ledger Logic: Ending a session mathematically restricts and triggers automatic valid transaction debits based on usage, validating > ₹0 balances before initiation.

Implementation Summary

The system robustly enforces data boundaries avoiding any overlapping reservations and prevents offline errors via standardized PWA protocols. Anticipated future iterations include expanding generic device camera parsing capabilities and integrating direct financial gateway API pipelines to productionize the internal wallet system.
